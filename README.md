## 获取工具

`lavas` 提供了一个命令行工具，可以通过它来快速创建 PWA 项目。它提供了多种常用的 APP Shell 给用户选择，降低了开发成本，也简化了工作流程，让 PWA 项目变得易于管理。可以通过 `npm install` 方便快捷地安装 lavas：

**安装**

```bash
npm install -g lavas
```

**使用**

```bash
lavas init
```
